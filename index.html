Genetic Learning Simulation

This small Javascript based application simulates evolution on a small scale. It demonstrates the fragility of nature with the use of a genetic learning algorithm. In this simulation, only the strongest creatures survive the natural disasters.

Cave Run is a Javascript based sidescroller. You run through a cave shooting at monsters that attack you. Currently working on my Github with Javascript. I have a mobile port built in unity that I never published to app store but is working as well.
